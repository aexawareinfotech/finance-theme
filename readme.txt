A professional WordPress theme for finance and loan companies by Aexaware Infotech.

== Description ==

Finance Theme is a clean, professional WordPress theme designed for finance companies, lenders, and loan providers. It features a mobile-first responsive design, compliance-ready footer sections, and customizable components for showcasing loan products.

**Developed by Aexaware Infotech**
https://aexaware.com

== Features ==

* Mobile-first responsive design
* Customizable hero section with loan calculator
* Loan types showcase slider
* Process steps section
* Testimonials display
* FAQ accordion
* Blog section
* Compliance footer (ASIC/AFCA ready)
* Theme Customizer options
* Translation ready
* Auto-updates via GitHub

== Installation ==

1. Upload the theme folder to `/wp-content/themes/`
2. Activate the theme through 'Appearance' > 'Themes' menu in WordPress
3. Go to 'Appearance' > 'Customize' to configure theme options
4. Navigate to 'Appearance' > 'Theme Setup' to import demo content (optional)

== Auto Updates ==

This theme supports automatic updates from GitHub. Updates are checked periodically and you will be notified when a new version is available.
